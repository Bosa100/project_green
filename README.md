# Project "Green"

Green Lantern Green House Project
This project is based on analyzing and monitoring certain subsets of data including:
1. Humidity
2. Temperature
3. Moisture
4. Light
5. UV Index

Using this data, we will be able to automate certain *tasks* accordingly.
